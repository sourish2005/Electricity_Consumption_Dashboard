# Electricity Bill \& Energy Consumption Analysis

## Project Overview

This project analyzes electricity bill and energy consumption data using Microsoft Excel. It demonstrates data cleaning, transformation, analysis, and dashboard creation to provide meaningful insights into energy usage patterns.

## Objectives

* Analyze electricity consumption data.
* Clean and organize raw datasets.
* Build an interactive dashboard.
* Identify consumption trends and key performance indicators (KPIs).
* Support data-driven decision making.

## Features

* Data Cleaning
* Pivot Tables and Pivot Charts
* Interactive Dashboard
* Slicers and Filters
* KPI Summary
* City-wise and Monthly Analysis
* Dynamic Reports

## Dataset

The dataset contains electricity consumption and billing information, including monthly energy usage, appliance-related consumption, operating hours, and city-wise records.

## Dashboard

The dashboard provides:

* Total electricity consumption
* Monthly trends
* City-wise comparison
* Appliance-wise analysis
* Interactive filtering
